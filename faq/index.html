Frequently Asked Questions

General

What is BTD5 Restored?
BTD5 Restored is a fan-made recreation of Bloons Tower Defense 5, made in Godot 4.
Is this an official release?
No. This is a fan project and is not affiliated with or endorsed by Ninja Kiwi.
Why did you make this?
The original Flash version can only* be played on Ninja Kiwi Archive, has sub-par performance, and all of the features that required a server don't work anymore. This project aims to restore (and improve) that functionality.

Legal / Copyright

Is this legal?
I'm not a lawyer. This is just a fan project that aims to preserve gameplay. I do not make any money off of it. If you're the rights-holder and have concerns, please contact me (see "Contact & DMCA" below) and I'll respond.
Do you use original assets?
No. The assets used in BTD5 Restored are from the original Flash version of BTD5. They have been ripped from the original .swf file using JPEXS Free Flash Decompiler.
Trademark / Credits
"Bloons Tower Defense 5" and related marks are the property of their respective owners. This game is a fan project and credits the original creators throughout the site.

Gameplay & Features

Which features are restored?
As of now, bloon movement, rounds, and a few basic towers have been ported. In the future, I plan to expand to almost (if not all) of the original game's content.
Are there microtransactions or paid content?
No. The game does not implement or require the original in-app purchase systems, and it never will. Any feature that relied on spending real money (such as "NK Coins") will be replaced with a free alternative that can be earned in-game.

Bugs, Reporting & Changelog

How do I report a bug?
Please open an issue using the GitHub issue tracker. Consider checking if the issue has already been reported to avoid duplicates.
Where can I see the updates?
Check the Changelog page and GitHub releases or commits for logs of fixes, features, and builds.
